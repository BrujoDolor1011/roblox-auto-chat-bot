Roblox Auto Chat Bot

Un bot automático para Roblox que envía mensajes y anuncios en intervalos de tiempo definidos, además de prevenir el estado de inactividad (AFK). Está diseñado para interactuar con los jugadores en el juego Pls Donate. La aplicación también cuenta con una interfaz gráfica personalizable para gestionar el comportamiento del bot.

Características

✅ Envía mensajes automáticos en intervalos de tiempo configurables.
✅ Previene la desconexión por inactividad (anti-AFK).
✅ Interfaz gráfica con CustomTkinter para fácil configuración.
✅ Atajo de teclado (Shift + P) para pausar y reanudar el bot.
✅ Soporte para temas claro y oscuro.

Instalación

Clona el repositorio:

git clone https://github.com/tu-usuario/roblox-auto-chat-bot.git
cd roblox-auto-chat-bot

Instala las dependencias necesarias:

pip install -r requirements.txt

Ejecuta el script:

python main.py
